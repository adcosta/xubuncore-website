Xubuncore: Xubuntu 12.04 LTS + Core 4.6

Para uso nas aulas práticas aqui se disponibiliza uma máquina virtual Linux Ubuntu 12.04 LTS com o emulador CORE 4.6 instalado e mais alguns extras. Incluem-se as instruções que foram usadas na sua criação.

1. Imagem XubunCORE (Xubuntu + CORE)

  • Versão VirtualBox: XubunCORE_VBOX.zip (2.99 GB zipped, 6.96GB unzipped) (Fevereiro 2015)

    Máquina virtual em formato original VirtualBox (para quem usa VirtualBox, apenas). Xubuntu 12.04 LTS + Core 4.6 (1GB Memória, 1 Processador, 20GB Disco, Intel PRO/1000 em NAT)

  • Versão VMWare: XubunCORE_VMWARE.zip (4.01 GB zipped, 8.64 GB unzipped) (Fevereiro 2015)

    Máquina virtual em formato original VMWare, compatível com VMWare Workstation 10.0. Xubuntu 12.04 LTS + Core 4.6 (1GB Memória, 1 Processador, 20GB Disco (max), Network Adapter: NAT, Teclado português, "VMware Tools" instaladas)

2. Instruções usadas

Em resumo, as imagens VMWare e Virtualbox acima disponibilizadas foram preparadas da seguinte forma:

  • Instalação normal do sistema operativo (X)Ubuntu 12.04 LTS (recomendado)
  • Instalação dos packages de suporte (pre-requisitos CORE)
  • Download e instalação do CORE 4.6 (servidor e ambiente gráfico)
  • Instalação de packages extra necessários para os trabalhos práticos

Em detalhe, os passos acima enumerados traduzem-se nas seguintes instruções:

  1. Descarregar e instalar Xubuntu 12.04 LTS (instalação normal ou vmware)

    Q: Porquê Xubuntu?
    R: Pela enorme popularidade da distribuição Ubuntu, pela estabilidade da versão 12.04 TLS, e porque usa um ambiente gráfico Desktop menos exigente em termos de recursos (memória e CPU): o XFce. Melhor só o Lubuntu 12.04 LTS

  2. Atualizar packages: 
    $ sudo apt-get update
$ sudo apt-get dist-upgrade
  3. Instalar packages de suporte (pré-requisitos): 
    $ sudo apt-get install bash bridge-utils ebtables iproute libev-dev libtk-img python tcl8.5 tk8.5 xterm
$ wget http://downloads.pf.itd.nrl.navy.mil/ospf-manet/quagga-0.99.21mr2.2/quagga-mr_0.99.21mr2.2_i386.deb
$ sudo dpkg -i quagga-mr_0.99.21mr2.2_i386.deb
  4. Instalar CORE: 
    $ wget http://downloads.pf.itd.nrl.navy.mil/core/packages/4.6/core_4.6-0ubuntu1_precise_i386.deb
$ sudo dpkg -i core_4.6-0ubuntu1_precise_i386.deb
  5. Instalar alguns packages extra que podem ser úteis: 
    $ sudo apt-get install wireshark tcpdump iperf wget lynx net-tools
$ sudo apt-get install vsftpd atftpd openssh-server
$ sudo apt-get install dnsutils bind9
  6. Testar: 
    $ sudo /etc/init.d/core start
$ sudo core
