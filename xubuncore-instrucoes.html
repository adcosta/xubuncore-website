Editado em: 13.fevereiro.2024
NOTA: Estas instruções só devem ser usadas se só se pretender recriar uma máquina virtual de raíz igual à que está disponível para download.

Instruções que foram usadas para criar as VMs Xubuncore

Em resumo, as imagens VMWare e Virtualbox acima disponibilizadas foram preparadas da seguinte forma:

  • Instalação normal do sistema operativo (X)Ubuntu 20.04 LTS (recomendado)
  • Instalação dos packages de suporte (pre-requisitos CORE)
  • Download e instalação do CORE 7.5.2 (servidor e ambiente gráfico)
  • Instalação de packages extra necessários para os trabalhos práticos

Em detalhe, os passos acima enumerados traduzem-se nas seguintes instruções:

  1. Descarregar e instalar Xubuntu 20.04 LTS

    Q: Porquê Xubuntu?
    R: Pela enorme popularidade da distribuição Ubuntu, pela estabilidade da versão 20.04 TLS, e porque usa um ambiente gráfico Desktop menos exigente em termos de recursos (memória e CPU): o XFce. Melhor só o Lubuntu 20.04 LTS

  2. Atualizar packages: 
    sudo apt-get update 
sudo apt-get upgrade
  3. Instalar/atualizar packages de suporte: 
    sudo apt install build-essential
sudo apt install virtualbox-guest-dkms virtualbox-guest-x11
sudo apt install git gcc make vim libtk-img bash xterm
sudo apt install sntp ntp wget lynx curl net-tools traceroute tcptraceroute 
sudo apt install ipcalc socat hping3 httpie whois ngrep
sudo apt install tcpdump wireshark iperf iperf3 tshark openssh-server openssh-client openssh-sftp-server
sudo apt install vsftpd atftp atftpd apache2 mini-httpd openvpn isc-dhcp-server isc-dhcp-client
sudo apt install bind9 bind9-utils
  4. Instalar CORE: 
    cd 
git clone https://github.com/coreemu/core.git
cd core 
sudo ./install.sh -v -p /usr/local
  5. Testar o funcionamento do CORE

    Iniciar o daemon: 

    sudo systemctl start core-daemon.service

    Ativar automaticamente no arranque (boot): 

    sudo systemctl enable --now core-daemon.service

    Correr o CORE: 

    sudo core-gui
